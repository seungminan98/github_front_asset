## Tab 4 — 한국 주거 전략 계산기

### 입력 파라미터
- 총 자산 (만원)
- 월 수입 세후 (만원)
- 선호 지역 (드롭다운)
- 주거 형태 선호 (아파트/빌라/오피스텔)
- 평수 선호 (20평/25평/30평/35평)
- 주거 우선순위 (통근편의 / 학군 / 가성비)

### 지역별 시세 데이터 (2024-2025 기준, 25~30평 아파트)
| 지역 | 매매(억) | 전세(억) | 월세(보증+월, 만원) |
|------|---------|---------|-------------------|
| 서울 강남 | 25~35 | 12~18 | 5000+300~400 |
| 서울 잠실 | 18~28 | 10~15 | 5000+250~350 |
| 서울 양재/서초 | 20~30 | 11~16 | 5000+270~370 |
| 경기 판교 | 12~18 | 6~10 | 3000+150~220 |
| 경기 수지 | 8~13 | 4~7 | 2000+100~160 |
| 경기 광교 | 9~14 | 5~8 | 2000+110~170 |
| 경기 영통 | 6~10 | 3~6 | 1000+80~130 |
| 경기 오산 | 3~6 | 2~4 | 500+50~90 |
| 경기 동탄 | 7~12 | 4~7 | 1000+90~150 |
| 경기 고덕 | 6~10 | 3~6 | 1000+80~130 |

### 판단 로직
1. **매매 가능 여부**: 총자산 × 0.4 이상 + 대출 가능액 (DSR 40% 적용)
   - 월 대출 가능: 월수입 × 0.4
   - 대출 한도: 월상환액 ÷ (연이율5% ÷ 12) × (1 - (1+r)^-360)
2. **전세 vs 월세 비교**
   - 전세 이자비용: 전세금 × 3.5%/12 (월)
   - 월세: 보증금 기회비용 + 월세액
   - 손익분기: 전세금 × 3.5% vs 월세 × 12
3. **투자 시드 = 총자산 - 주거비용(보증금)**
4. **업계 표준 기준**:
   - 주거비 권장: 월수입의 25~30% 이하
   - 대출 원리금: 월수입의 40% 이하 (DSR)
   - 비상금: 월수입 × 6개월치 확보 필수

### 총자산별 추천 시나리오
| 총자산 | 월수입 | 추천 전략 |
|--------|--------|---------|
| 1억 미만 | ~250만 | 월세 + 전액 적립식 ETF |
| 1억 미만 | 300만+ | 경기 외곽 전세 + 소액 ETF |
| 1~2억 | ~350만 | 경기 전세 (대출병행) + S&P500 ETF |
| 1~2억 | 400만+ | 경기 전세 자기자본 + 채권+주식 분산 |
| 2~4억 | ~400만 | 경기 전세 or 소형 매매 검토 + 분산투자 |
| 2~4억 | 500만+ | 서울 외곽/경기 매매 + 풀 포트폴리오 |
| 4억+ | ~500만 | 서울 매매 가능 + 여유자금 투자 |
| 4억+ | 600만+ | 강남권 전세 or 중간권 매매 + 복합투자 |

## 구현 순서
1. HTML 뼈대 + CSS (뉴모피즘 스타일 시스템)
2. Tab 1: 역사 타임라인 (Chart.js + 이벤트 토글)

6. Tab 5: 현재 포지션 (정적 + 도넛 차트)
